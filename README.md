# Holidays-API-App

A responsive frontend web application that fetches and displays public holiday data from a **REST API**, allowing users to view and filter holidays based on scope.

---

## 📌 Project Overview
This project demonstrates core frontend skills including API consumption, dynamic UI rendering, and client-side filtering using vanilla JavaScript.

The application retrieves holiday data from a public API and dynamically renders the results on the page based on user-selected criteria.

---

## 🧠 Features
- Fetches holiday data from a **REST API**
- Dynamically displays holiday information on the page
- Boolean-based filtering for **nationwide vs. regional** holidays
- Responsive layout for desktop and mobile devices
- Clean and simple user interface

---

## 🛠 Tech Stack
- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **Bootstrap**
- **REST API**

## ⚙️ How It Works
- Uses the JavaScript `fetch()` API to retrieve holiday data in JSON format
- Parses API responses and dynamically creates DOM elements
- Updates the displayed content based on user-selected filters
- Handles empty results gracefully
